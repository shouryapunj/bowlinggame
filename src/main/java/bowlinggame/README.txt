# BowlingGame

BowlingGame is built on Java using Springboot framework.

## How to run
1) Using IDE - import Project in an IDE such as IntelliJ/Eclipse/STS and run the main method from Application.java
2) Using command line (maven required) -
    Navigate to BowlingGame folder and run -
    2.1) mvn package
    2.2) mvn spring-boot:run

## APIs
1) localhost:8080/start - Starts a new game on the prompt
2) localhost:8080/scoreboard - Shows the game state in JSON format
